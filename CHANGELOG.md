# 更新日志

所有针对当前软件功能和 UI 界面的重要更改都会记录在此文件中。

## 未发布版本

### 功能改进
- **双语合并：** 重构双语字幕合并功能。支持自定义翻译字幕后缀及合并后的输出后缀，支持自定义两种语言的 ASS 字幕样式名称与详细定义，同时允许自定义作者（Author）和注释（Comment）等信息。修改配置时会自动保存（至 `config.json`），并在下次启动时自动加载。功能相关的 UI 文本提示也进行了局部优化。
- **帧率转换：** 新增字幕帧率转换功能，支持在各大常见帧率间互转（如 23.976 <-> 25），自动重算并在新旧帧率间同步时间轴。 (#fbe7114)
- **格式支持 (WebVTT)：** 现支持在字幕提取时识别 `webvtt` 轨道并为其提供内置的“转为SRT”勾选转换选项。 (#5bf33d2)

### 移除
- **术语合并：** 移除了“术语合并”相关的功能、UI 和底层逻辑。 (#77da965)

## 0.9.2

### 功能改进
- **字幕处理：** 优化了中文 SRT 字幕的规则格式化操作。 (#92c9a8e)
- **拖放响应：** 改进了应用的多文件拖拽解析逻辑，确保能更健壮地识别有效文件路径，并增加了空输入的检测防错处理。 (#3cc04db)

### 界面调整 (UI)
- **提取进度：** 细化了“字幕提取”界面中的处理进度日志显示，提供了更友好的用户反馈。 (#8a1e132)
- **布局整合：** 优化了主界面基础框架，使用网格布局（Grid）整合了各项功能的按钮与拖放区域，并将“跳过合并”复选项内置到对应处理区以节省空间。 (#78b803d)

---

## 0.9.1

### 功能改进
- **字幕提取 (MKV)：** 新增对未知/异常字幕流编码的 Fallback（回落）支持，自动调用 `mkvmerge` 与 `mkvextract` 接管底层的解析及提取。 (#23c98b4)

---

## 0.9.0

### 功能改进
- **核心功能提取：** 首次推出针对 MP4 / MKV 视频的内部独立字幕流解析与提取功能。 (#5bddfab)
- **批量处理：** 全面支持各类文本以及多文件的批量式拖拽合并，并改进了合并执行时的日志描述。 (#5128802)
- **程序版本识别：** 引入内置的版本控制机制，并在窗口标题栏实时提供当前版本信息展示。 (#c067db8)
